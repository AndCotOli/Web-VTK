# Usage

```
# Create a Virtual Environment
python -m venv .venv
source .venv/bin/activate

# Install dependencies
pip install trame

# Deactivate the venv
deactivate

# Run the app with Paraview python
~/../../opt/paraview/bin/pvpython src/main.py --venv .venv --server
```
